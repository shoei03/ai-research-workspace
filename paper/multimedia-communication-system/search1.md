## アプリ1：Rakuten Link（楽天モバイル）

- 公式ページ：https://network.mobile.rakuten.co.jp/service/rakuten-link/
- cite キー：`rakuten-link-service`

### サービス概要

楽天モバイル（Rakuten最強プラン）の契約者が利用できる通話・メッセージアプリ。
自分の携帯番号（070/080/090）から、一般の固定電話・携帯電話（他社含む）へ発信可能。

### 料金体系

| 条件 | 料金 |
|---|---|
| Rakuten Link から国内通話（固定・携帯問わず） | **無料** |
| Rakuten Link を使わずOS標準アプリから | 30秒22円 |
| 楽天Link同士のSMS | 無料（Android版） |
| iOS版SMS | OS標準アプリ経由のため別途料金 |

※ Rakuten最強プランの月額は3GB以下1,078円・20GB超3,278円（通話アプリは別途無料）。
- 一次情報源：`rakuten-link-service` / `rakuten-link-faq-billing`

### 通話品質

公式ページには音声品質の技術仕様（コーデック等）の明示はない。

公式FAQに記載された品質関連の注記：
- **Wi-Fi接続時は動作保証外**（「Wi-Fiは設置環境等によって電波の状況が異なり、通話品質が不安定になることがあります」）
- LTE接続での利用を推奨
- VPN接続時は動作保証外
- デュアルSIM設定時は回線自動切り替えにより品質影響の可能性
- 一次情報源：`rakuten-link-faq-quality`

### メリット（公式が宣伝している内容）

1. 「ご自身の電話番号から、ほかの携帯電話会社ケータイや固定電話を含む国内への通話が無料でかけられます」
2. Rakuten Link を使った通話はデータ容量にカウントされない
3. 「海外の対象国と地域からであれば、日本国内へは無料で電話がかけられます」（海外滞在中の発信）
4. 連絡先・通話履歴の自動バックアップ、機種変更時にデータ復元可能
5. 楽天ポイント還元・キャンペーン特典との連携

- 一次情報源：`rakuten-link-service`

### デメリット・利用制限

1. **楽天モバイル契約が前提条件**（単独では利用不可）
2. **無料対象外番号**（自動でOS標準アプリに切り替わり有料）：
   - 0570（ナビダイヤル）
   - 171（災害伝言ダイヤル）
   - 188（消費者ホットライン）
   - \#7119, \#8000, \#8008, \#8011, \#8080, \#8139, \#8778, \#8891, \#9110 等
   - 一次情報源：`rakuten-link-faq-outofscope`
3. **iOS版の着信制限**：「Rakuten Linkでは着信できません。iOS標準の電話アプリでの着信となります」
4. **Wi-Fi・VPN・Bluetooth 接続時は動作保証外**
5. **SMS送信上限**：最大200通/日
6. **長時間・大量通信時の制限**：「当社設備に影響をおよぼすと判断した場合は切断することがある」「一定期間内に大量・多数の通信があった場合、利用を制限または中止することがある」
7. **Webサイトのリンクから発信した場合** や **着信履歴からの折り返し**（OS標準アプリ）は有料

## アプリ2：G-Call 050（株式会社ジー・コミュニケーション）

- 公式ページ：https://www.g-call.com/050/
- cite キー：`gcall050-service`

### サービス概要

050番号を取得して使うIP電話スマホアプリ。楽天モバイルを含む主要キャリア・MVNO回線で利用可能。
既存の携帯番号とは別に050番号が付与される。

### 料金体系

| 項目 | 料金 |
|---|---|
| 月額基本料 | **308円**（税込） |
| 国内携帯電話への発信 | **1分 17.6円**（税込） |
| 国内固定電話への発信 | **3分 8.8円**（税込） |
| G-Call 050同士 | **10分まで無料** |
| 国際通話（米国） | 1分 5円 |
| 国際通話（中国・韓国・シンガポール） | 1分 10円 |

- 一次情報源：`gcall050-service`（https://www.g-call.com/050/index.php?pg=2）

#### 参考：標準キャリアとの比較

| 発信先 | G-Call 050 | 標準キャリア（目安） |
|---|---|---|
| 国内携帯 | 17.6円/分 | 22円/30秒（約44円/分） |
| 国内固定 | 8.8円/3分 | 132円/3分 |

### 通話品質

インターネット接続（4G/5G/Wi-Fi）を使うVoIP通話。品質は回線状況に依存。

公式記載：「ご利用の地域やネットワーク環境によって通話品質が変動する場合がある」
現代の4G/5G環境では実用上十分な品質とされる（ITレビュー等）。
コーデック等の技術仕様の公式開示なし。

- 一次情報源：`gcall050-service`

### メリット（公式が宣伝している内容）

1. 月308円の低コストで050番号を取得できる
2. 携帯電話通話料が標準の約60%オフ、固定電話は約93%オフ
3. どのキャリア・MVNO（ドコモ・au・ソフトバンク・楽天等）でも利用可能
4. **海外滞在中でも国内と同料金で日本に発信可能**（データ通信さえあれば）
5. 留守番電話機能付き（メール通知）
6. アプリがバックグラウンドや非起動時もプッシュ通知で着信

### デメリット・利用制限

1. **緊急通報（110・118・119）への発信不可**（IP電話の制度的制限）
2. インターネット接続必須（通信障害時はそもそも利用不可）
3. 050番号のため相手側に「050-xxxx-xxxx」と表示される（携帯番号から電話したい場合には不向き）
4. **フリーダイヤル（0120・0800）等への発信不可**
5. 接続協定未締結事業者への通話不可
6. 月間通話料が一定額を超えた場合に利用停止の可能性
7. 機能制限：通話転送・保留・暗号化（SSL/TLS）等は非対応
